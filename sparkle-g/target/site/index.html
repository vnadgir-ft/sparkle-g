Introduction

The SPARQL query language for RDF is designed to meet the use cases and requirements identified by the RDF Data Access Working Group in RDF Data Access Use Cases and Requirements; detailed explanations can be found on the official W3C's SPARQL specifications

Purpose of this project is provide a cross-compiler ANTRL v3 grammar which is an implementation of the SPARQL grammar specifications.
